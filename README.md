# FSD-task
Бронирование номеров в лучшем отеле 2019 года по версии ассоциации «Отельные взгляды»

[Демо](https://hoka-hoka.github.io/multirange/ "Необязательная подсказка")
### Установка

Установка зависимостей  
`npm i`

Разработка  
`npm run dev`

Сборка  
`npm run build`
